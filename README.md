# Europass [![Build Status](https://drone.io/github.com/jorgeb1980/Europass/status.png)](https://drone.io/github.com/jorgeb1980/Europass/latest)
===========

Java library that renders a PDF version of the information expected for a Europass format resumé.

See https://europass.cedefop.europa.eu/es/documents/curriculum-vitae/templates-instructions
